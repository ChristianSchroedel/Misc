JavaSamples
===========

Small Java applications

Each folder contains a java example with source files

FirstWindow
-----------
Create your first window with Java

WindowRandomColor
-----------------
Click on the window to switch its background color

InchConverter
-------------
Convert centimeters into inch and backwards

HexEditor
---------
Open a file and edit its content (all in hex)

ThreadsVar1
-----------
Create/start/stop a thread

ThreadsVar2
-----------
Create a thread and start/pause it

Drawing
-------
Draw a clock on a drawing pane

DrawingMVC
----------
Create a little paint application

Chat
----
Use network sockets to chat with each other

CountdownBean
-------------
Add your own bean/widget to an application

myLibs
------
Create a library project
